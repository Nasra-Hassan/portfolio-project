# My One-Page Portfolio

## Description

This is a simple **one-page portfolio website** built using **HTML and CSS only**.
It showcases my basic web development skills and includes sections like About, Projects, and Contact.

---

## Features

* Single-page layout (no multiple pages)
* Responsive design (works on mobile, tablet, and desktop)
* Clean and simple user interface
* Beginner-friendly code structure

---

## Built With

* HTML5
* CSS3

---



## Sections Included

* **About** – Brief description about me
* **Projects** – Displays my work (e.g., calculator app)
* **Contact** – My contact information
